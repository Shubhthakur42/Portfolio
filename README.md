## Run Locally

From the project root:

```bash
npm run install-all
npm run start
```

Other useful scripts:

```bash
npm run client
npm run server
npm run build
```

## Environment Setup

- Client environment templates are in client/.env.example
- Server environment templates are in server/.env.example

Create your .env files before running production integrations.
